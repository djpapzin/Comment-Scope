I'm excited to share my new Streamlit app, "YouTube Comment AI Scrutinizer"! 

This app uses Google's Gemini AI Flash model to analyze YouTube comments and provide insightful data visualizations. 

**Features:**
- Sentiment analysis
- Word cloud generation
- Comment length analysis
- Top commenters
- Sentiment analysis over time
- Interactive data table
- User engagement score
- Comment summarization
- Trending video analysis

**Check out the demo:** [https://youtube-comment-ai-scrutinizer.streamlit.app/](https://youtube-comment-ai-scrutinizer.streamlit.app/)
